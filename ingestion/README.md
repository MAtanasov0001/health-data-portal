# Приемна тръба (ingestion)

Режим **B** (засилено ревю). Python пакет, който превръща подаден файлов набор в **неизменяем,
версиониран снапшот**, готов за каталога и публичното API.

## Тръба

```
вход (CSV + metadata.yaml + schema.yaml)
  → 1. схемна валидация (Pydantic)         ── невалиден вход се отхвърля
  → 2. контрол на разкриването             ── small-cell suppression (min размер на клетка)
  → 3. неизменяем версиониран снапшот       ── SHA-256 checksum + времеви печат ISO 8601 / UTC+2
  → 4. DCAT-AP запис за набора               ── влиза в каталога
```

Всяка стъпка оставя **одитен запис по МЕ64** (`ingestion/audit.py`) в централизиран JSON Lines
дневник `snapshots/audit/ingest-audit.jsonl` (само за добавяне; удобен за SIEM). Всеки запис носи
деветте задължителни атрибута: уникален номер, точно време (ISO 8601 / UTC+2), вид по номенклатура,
данни за системата, идентификатор на компонента, приоритет, описание, данни за събитието и резултат.
Отхвърляне по схема се записва с приоритет `error` и резултат `failure`.

## Нормативни връзки

- **Контрол на разкриването** — независимата защита срещу реидентификация (ядро на правната
  защита; концепция v2.0). Клетки под праг се потискат, вкл. вторично потискане, за да не се
  възстановяват чрез суми.
- **Времеви печати** по ISO 8601, часова зона UTC+2 (Europe/Sofia), от сървъра (чл. 46).
- **Версии** — всеки снапшот е неизменяем и версиониран; предходните остават достъпни (чл. 14).
- **Контролна сума** — SHA-256 върху съдържанието на снапшота за проверимост.

## Употреба

```bash
pip install -e ".[dev]"

# Пусни пилота през тръбата:
python -m ingestion.pipeline run ../pilot/hospitalizacii-po-oblast-2023

# Валидирай DCAT-AP пример спрямо профила:
python -m ingestion.dcat.validate ../spec/dcat-ap/examples/dataset.jsonld
```

Снапшотите се записват в `snapshots/<identifier>/<version>/` (извън git — виж `.gitignore`).

## Тестове

```bash
pytest -q       # обхваща схема, контрол на разкриването (вкл. вторично потискане), снапшот
ruff check . && ruff format --check .
mypy ingestion
```
